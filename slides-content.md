## 📅 Meeting 05 Jul 2026 — การบ้านแต่ละฝ่าย

### 1. ฝ่ายออกแบบเส้นทางธรรมชาติ
**เป้าหมาย**: เส้นทางที่ "สนุก ปลอดภัย มีเรื่องราว และได้เรียนรู้ธรรมชาติ"

การบ้าน:
- Mapping จุดเด่นธรรมชาติ: จุดวิว / จุดเรียนรู้ / จุดพัก / จุดถ่ายรูป / จุดเสี่ยง
- กิจกรรมระหว่างทาง: Nature checkpoint / ภารกิจ / Sound installation / Storytelling station
- ธีมแต่ละช่วงเส้นทาง, วิเคราะห์ระดับความยาก
- เวลาที่เหมาะสม: วิ่งเช้า / กลางคืน / Sunrise / Sunset

คำถามต้องตอบได้: ทำไมเส้นทางนี้ต่างจากงาน Trail อื่น / จุด WOW คืออะไร / นักวิ่ง "รู้สึกอะไร" หลังจบงาน

**แผนลงพื้นที่ครั้งแรก (วังหมี กระทิง)**:
- เส้นทางวิ่ง trail 15 กม. + เส้นศึกษาธรรมชาติ 5 กม.
- หาจุดอุปสรรค, จุดถ่ายภาพ, จุดเรียนรู้ธรรมชาติ, จุดสื่อความหมายธรรมชาติ
- จุดให้น้ำ + พยาบาล + checkpoint อยู่ที่เดียวกัน ทุก 3-4 กม.
- จุดป้ายบอกทาง

**งบสำรวจ 2 วัน 1 คืน (นอนเต็นท์)**:
- สมาชิก: จอย ทัช มี่ หนุ่ย พร นัน อิง
- ค่าน้ำมัน Eco Car 2 คัน ไป-กลับ 498 กม. ≈ 1,800–2,600 บาท (สรุปที่ 2,600)
  - ราคาน้ำมัน ณ 8 ก.ค. 2569: แก๊สโซฮอล์ 95/ดีเซล ~34.94 บ./ล., E20 ~29.94 บ./ล.
- ค่าอาหารเหมา (ซื้อของทำกินเอง) 1,000/วัน × 2 = 2,000 บาท
- **รวมทั้งหมด 5,000 บาท**

### 2. ฝ่ายกิจกรรมและ Event
**เป้าหมาย**: สร้าง "Festival Experience" ไม่ใช่แค่งานแข่งขัน

การบ้าน:
- Theme งาน, Timeline กิจกรรม 2–3 วัน
- Concept เส้นทาง เช่น Forest Discovery Trail / Sunset Trail / Night Jungle Experience
- ระยะวิ่งเบื้องต้น: Fun Run / 10K / 25K / Ultra
- กิจกรรมเสริม: ดนตรีสด / Acoustic campfire / Workshop ธรรมชาติ / Stargazing / Yoga-Breathwork / Local food market / Outdoor cinema
- Opening/Closing Experience, จุดกิจกรรมครอบครัว-เด็ก
- ระบบ badge / mission / passport
- ประสบการณ์กลางคืน: Night walk / Glow trail / Silent disco
- Photo Moment / Art installation

คำถาม: ถ้าคนไม่วิ่งจะยังอยากมางานไหม / อะไรคือ "Moment ที่คนจะจำ"

### 3. ฝ่ายการตลาดและการสื่อสาร
**เป้าหมาย**: ทำให้งาน "มีตัวตนชัด" คนอยากพูดถึง

การบ้าน:
- Target Audience: นักวิ่งจริงจัง / สายแคมป์ / สายเที่ยวธรรมชาติ / คนเมือง / ครอบครัว / สาย content
- Branding direction (Mood/Tone/Key visual/Tagline), Story ของงาน
- แผน Content 3 เดือนก่อนงาน, ช่องทาง: FB / IG / TikTok / Running & Outdoor community
- Influencer/Partner, Viral idea, Community ก่อนวันงาน (Training run / Online challenge)
- วิเคราะห์คู่แข่งและจุดแตกต่าง

**Positioning (เสนอโดยกวาง)**: **Family Nature Trail Festival**
- ไม่ position เป็นแค่งานวิ่งเทรล — ถ้าแข่งด้วยระยะ/เหรียญจะถูกเทียบกับงานอื่นทันที
- ใช้การวิ่งเป็นประตูพาครอบครัว-เด็กกลับมาเชื่อมโยงธรรมชาติ → สร้างพื้นที่เรียนรู้ธรรมชาติระยะยาว
- หลักคิด: *"เราไม่คิดจะขายงานนี้ให้ทุกคน เราจะสร้างงานให้คนกลุ่มหนึ่งรักมันมากพอ จนเขาเล่าให้คนอื่นฟังเอง"*
- ❌ ไม่ทำภาพงานวิ่ง Hardcore / ❌ ไม่พูดแบบ organizer ประกาศแข่งขัน — พูดแบบ "คนชวนกันออกไปค้นพบธรรมชาติ"
- Visual: เด็ก / ครอบครัว / เท้าเดินบนดิน / มือสัมผัสต้นไม้ / แสงเช้า / หมอก / การค้นพบ / Moment เล็กๆ
- สูตร Content: **40% Story / 30% Experience / 20% Community / 10% Direct Sales**
- งานนี้ขาย: ความรู้สึก + การค้นพบ + การมีส่วนร่วม

**Concept**: WANG MI — TRAIL OF THE FUTURE
> "เส้นทางนี้ยังสร้างไม่เสร็จ เพราะเรากำลังสร้างมันไปด้วยกัน"

**Journey Experience 5 ขั้น**:
1. ARRIVE — ฉันมาถึงโลกของ Wang Mi
2. DISCOVER — ฉันค้นพบธรรมชาติและวังหมี
3. PLAY & CONNECT — ฉันทำกิจกรรมกับครอบครัวและผู้คน
4. ACHIEVE — ฉันพิชิตบางอย่าง
5. LEAVE A LEGACY — ฉันทิ้งบางอย่างที่ดีไว้ให้อนาคต

**Installation ideas**:
- แผนที่วังหมีขนาดใหญ่ ~3×5 ม. (ภูเขา ป่า ลำธาร ชุมชน ฟาร์ม เส้นทางวิ่ง ศูนย์เรียนรู้อนาคต) + จุด TODAY/FUTURE + เด็กติด sticker "ฉันอยากเห็นอะไรเกิดขึ้นที่นี่"
- **LEGACY WALL** — กำแพง "ฉันทิ้งอะไรไว้ให้อนาคต" (ฉันอยากให้เด็กในอนาคต… / ฉันอยากให้ธรรมชาติ… / วันนี้ฉันมาวิ่งเพราะ…)
- **Circular Infrastructure**: วัสดุ Gate (ไม้ ผ้า เชือก พืช โครงสร้าง Modular) นำไปสร้างศูนย์เรียนรู้หลังงาน

### 4. ฝ่ายระดมทุนและสปอนเซอร์
**เป้าหมาย**: พาร์ตเนอร์ที่ "เข้ากับจิตวิญญาณงาน"
- ประเภท sponsor: Outdoor brand / Energy drink / รถยนต์ / Camping / Sustainability / Health & wellness
- Package: Title / Main / Activity sponsor + สิทธิประโยชน์ + activation ในงาน
- วิเคราะห์มูลค่าที่ sponsor ได้, รายชื่อเป้าหมาย, pitch idea

คำถาม: งานสร้างคุณค่าอะไรให้ sponsor / sponsor จะ "ดูดี" ยังไงในงานนี้

### 5. ฝ่ายชุมชนและผู้มีส่วนได้ส่วนเสีย
**เป้าหมาย**: งาน "เติบโตไปกับพื้นที่"
- ระบุ stakeholder: ชุมชน / รัฐ / อุทยาน / ผู้ประกอบการ / เจ้าของพื้นที่
- ประโยชน์ชุมชน, การมีส่วนร่วมท้องถิ่น, local experience, ร้านค้า-อาหารท้องถิ่น
- วิเคราะห์ผลกระทบ, แผนสื่อสารกับชุมชน

คำถาม: ชุมชนได้อะไร / ทำยังไงให้งานถูกยอมรับระยะยาว

### 6. ฝ่ายปฏิบัติการและโลจิสติกส์
**เป้าหมาย**: งาน "ไหลลื่น" จัดการได้จริง
- Flow งาน, layout พื้นที่, การเดินทาง, parking/shuttle, น้ำ/ไฟ/ห้องน้ำ
- Registration, bag drop, จุดปล่อยตัว/finish, จัดการคนจำนวนมาก, bottleneck

คำถาม: จุดไหนเสี่ยง chaos / ถ้าฝนตกทำอย่างไร

### 7. ฝ่ายความปลอดภัยและการแพทย์
**เป้าหมาย**: ลดความเสี่ยงสูงสุด พร้อมทุกสถานการณ์
- วิเคราะห์ความเสี่ยง, จุดอันตรายบนเส้นทาง, แผน emergency, ระบบสื่อสาร
- Medical station, evacuation, แผนรับมืออากาศ/สัตว์ป่า, tracking นักวิ่ง, จำนวนทีมแพทย์/กู้ภัย

คำถาม: Worst case scenario / ใช้เวลากู้ภัยนานแค่ไหน

### 8. ฝ่ายสิ่งแวดล้อมและความยั่งยืน + คาร์บอนเครดิต
**เป้าหมาย**: ต้นแบบ "Green Trail Event" + คำนวณ CFP ของ event
- Zero Waste, คัดแยกขยะ, ลด single-use plastic, ระบบ refill, วัสดุย่อยสลายได้
- ลด carbon footprint, carbon offset/credit, ฟื้นฟูธรรมชาติหลังงาน, guideline นักวิ่ง, sustainability KPI, วิธีเก็บข้อมูล-คำนวณ carbon emission

**Green coach guideline**:
- อาหารท้องถิ่น: วัตถุดิบ 80% จากท้องถิ่น, ตัวเลือกพิเศษ (อิสลาม/มัง), สื่อสาร information
- บริจาค 10% ให้องค์กรความยั่งยืนชุมชน
- ประหยัดพลังงาน-น้ำ-ไฟ, พลังงานทดแทน, หลอด/หัวประหยัด, รถไฟฟ้ารับส่ง, ขนส่งสาธารณะ
- เป้า carbon = 0, รักษาพื้นที่สีเขียว, อนุรักษ์ความหลากหลายทางชีวภาพ
- จัดการขยะ: แก้วใช้ซ้ำ, รีไซเคิล, ฐานเรียนรู้แยกขยะ, เตาเผาขยะ, หนังสือแจ้งมลพิษทางเสียง
- การจ้างงานเท่าเทียม คุ้มครองสิทธิแรงงาน
- นำวัฒนธรรมท้องถิ่นเข้ามาเป็นกรอบ event
- สื่อสารให้ลูกค้าทราบความพยายามด้านความยั่งยืน + ข้อมูลสถานที่ธรรมชาติ (route ศึกษาธรรมชาติ)

**CFP plan**: ก่อนงาน / ระหว่างงาน (จุดแยกขยะ, บูทกิจกรรม, จุดมัดจำ-คืนขยะ) / หลังงาน
- Policy มัดจำขยะพลาสติก, เลือกของมี carbon tag, เลือกร้านค้าเป็นมิตรสิ่งแวดล้อม

### 9. ฝ่ายการเงินและบัญชี
**เป้าหมาย**: ควบคุมต้นทุน ยั่งยืนทางการเงิน
- ประมาณการรายรับ-รายจ่าย, fixed/variable cost, จุดคุ้มทุน, pricing strategy
- จำนวนผู้เข้าร่วมที่เหมาะสม, cash flow timeline, ความเสี่ยงการเงิน, contingency budget

คำถาม: ต้องมีคนขั้นต่ำเท่าไร / จุดที่งบบานปลายคืออะไร

### 10. ฝ่ายกฎหมายและเอกสาร
**เป้าหมาย**: ถูกกฎหมาย ลดความเสี่ยง
- ใบอนุญาต, เอกสารรัฐ, ประกันภัย, Waiver นักวิ่ง, สัญญาสปอนเซอร์/ผู้รับจ้าง
- PDPA/การใช้ภาพ, ข้อกำหนดพื้นที่ธรรมชาติ, ข้อจำกัดเสียง/เวลา

คำถาม: เรื่องไหนเสี่ยงผิดกฎหมายสุด / ต้องขออนุญาตล่วงหน้านานเท่าไร

### 11. ฝ่ายอาสาสมัครและบุคลากร
**เป้าหมาย**: ทีม "มีพลังและเข้าใจงาน"
- จำนวนคน, role, training, briefing, สื่อสารภายใน, สิทธิประโยชน์ volunteer, ดูแลทีม, จัดกะ, motivation

คำถาม: ทำยังไงให้ volunteer อยากกลับมาอีก / จุดไหนต้องใช้คนเก่งพิเศษ

### 12. ฝ่ายสรุปผลและประเมินผล
**เป้าหมาย**: วัดผลเพื่อพัฒนาระยะยาว
- KPI, feedback, survey, วัดผลเศรษฐกิจ/ชุมชน/สิ่งแวดล้อม/branding-social, บทเรียนหลังงาน, รูปแบบ report

คำถาม: งานสำเร็จเพราะอะไร / ปีหน้าพัฒนาอะไร
